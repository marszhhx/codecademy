CSS Position

Value Description
static Kept in the document flow
This is the default value
relative Relative position to its default position
Can be used to pair a relative parent with an absolute child
absolute Removed from the document flow
Relative to its closest "non-static" parent
fixed Removed from the document flow
Relative to the <html> element
sticky Keeps in the flow when scroll, but sticks to a specified position as the page is scrolled further. when scrolling, stops when hit the edge of the screen
